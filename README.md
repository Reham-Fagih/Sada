## Overview

SADA is an AI-powered financial digital twin platform currently under development that transforms personal financial data into an interactive virtual city experience.

The platform analyzes financial behavior, generates personalized insights, and visualizes financial health through a dynamic city that evolves based on user decisions.

SADA utilizes synthetic banking data to simulate realistic financial scenarios while developing an intelligent financial assistant that helps users understand their financial patterns and make informed financial decisions.

## Data & Simulation

SADA uses a synthetic financial dataset designed to simulate realistic banking scenarios without relying on real user financial information.

The dataset includes:

- Synthetic user personas
- Transaction histories
- Spending categories
- Financial goals
- Income and expense patterns
- Behavioral financial profiles

The dataset supports AI-driven analysis and enables the development of a dynamic financial city simulation.

## Impact

SADA aims to transform traditional financial tracking into an engaging and personalized experience by combining AI, financial analysis, and interactive visualization.

The platform helps users understand their financial behavior, explore future scenarios, and build better financial habits through an evolving digital city.
