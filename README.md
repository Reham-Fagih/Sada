## Overview

SADA is an AI-powered financial digital twin platform currently under development that reimagines the way individuals understand and interact with their financial future.

Despite the availability of many financial applications, users still struggle to understand their real financial situation and connect their daily decisions with their future outcomes.

SADA transforms complex financial data into an interactive experience by creating a personalized digital twin that analyzes financial behavior and visualizes its impact through a dynamic virtual city that grows with smart decisions and changes with unhealthy financial patterns.

## Synthetic Financial Dataset

SADA uses a fully synthetic financial dataset created to simulate realistic Saudi financial behaviors while maintaining user privacy.

The dataset includes:

- 10 synthetic financial personas
- 1,100+ financial transactions
- Six months of simulated financial activity
- Income, expenses, savings, and investment patterns
- User goals and financial scenarios

The dataset was generated using a Synthetic Data Generation Methodology to create realistic financial behaviors suitable for analysis and simulation.

---

## Data Processing Pipeline

The financial data workflow includes:

1. Generating realistic synthetic financial personas and transactions.
2. Validating and storing data using JSON and CSV formats.
3. Processing transactions into financial indicators.
4. Generating financial summaries and insights.
5. Visualizing results through the interactive financial city.
